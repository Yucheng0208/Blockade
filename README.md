# Install
## Python v3.x(and above)

## Pypi Step
* Step 1. Use the `pip install pip` function, install Pypi.
* Step 2. Use the `pip install numpy` function, install Numpy.
* Step 3. Use the `pip install tkinter` function, install Tkinter.
